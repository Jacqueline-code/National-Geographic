# National Geographic - Nature Explorer

A modern, responsive web experience showcasing nature storytelling with cinematic visuals, glass morphism design, and immersive interactions.

## Features

- **Multi-page Navigation**: Seamless navigation between Home, Stories, and Explore pages
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Dark/Light Mode**: Toggle between dark and light themes for comfortable viewing
- **Video Backgrounds**: Cinematic video backdrops on all pages
- **Interactive Elements**: 
  - Hover effects on images and cards
  - Flip cards in the Stories section
  - Smooth parallax scrolling effects
  - Menu animations
- **Glass Morphism UI**: Modern frosted glass effect containers with backdrop blur
- **Fully Responsive**: Mobile-first approach with breakpoints at 768px and 480px

## Pages

- **Home**: Featured stories with hero section and image grid
- **Stories**: Deep dive into the Amazon rainforest with flip cards for species information
- **Explore**: Discovery destinations with featured locations

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Grid, Flexbox, animations, glass morphism
- **JavaScript** - Scroll effects, theme toggle, menu interactions
- **Responsive Design** - Mobile-first with media queries

## How to Use

1. Open any `.html` file in your browser
2. Click the compass icon (☸) to open the navigation menu
3. Click the theme button (◐) to toggle between dark/light mode
4. Scroll to experience parallax effects
5. Hover over images for interactive effects
6. On the Stories page, click cards to flip and reveal information

## Browser Support

Works best on modern browsers (Chrome, Firefox, Safari, Edge) with CSS Grid and Flexbox support.
